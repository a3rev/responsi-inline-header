== Description ==

This Responsi Theme Framework plugin adds a single line header that has Logo | Nav Bar plus 2 optional widget areas.

= Features =

* The Inline header has 4 sections in a single row, Logo, Nav Bar plus 2 optional widget areas
* Set header width in px or full width of the screen
* Adjust each section width by percentage of total area of the header. 
* Header locks at the top of the screen on scroll
* Logo resizes on scroll
* In tablet and mobile menu shows as an icon and is positioned to the right of the screen
* 2 Layout options in mobile
